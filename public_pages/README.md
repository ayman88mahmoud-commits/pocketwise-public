# WalletBoard GitHub Pages Content

This folder contains static GitHub Pages content for WalletBoard:

- `index.html`
- `privacy.html`
- `support.html`

The pages are plain HTML with inline CSS. They have no external dependencies, no scripts, no analytics, no tracking, no images, and no logos.

## Publishing With GitHub Pages

1. Create a GitHub repository for the public pages, or use an existing repository intended for WalletBoard public content.
2. Upload the files in this folder to the repository.
3. In GitHub, open the repository settings.
4. Go to Pages.
5. Enable GitHub Pages for the branch/folder that contains these files.
6. Wait for GitHub Pages to publish.
7. Open the published site and confirm:
   - `index.html` loads.
   - `privacy.html` loads.
   - `support.html` loads.
   - Links between pages work.
   - The support email is correct.
8. Copy the final published URLs into the publishing docs and App Store Connect.
